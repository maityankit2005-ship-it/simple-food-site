# Delicio — Food Delivery (Static Frontend)

## Theme
Food Delivery / Restaurant listing — responsive landing + menu + contact form.

## Files
- index.html
- style.css
- images/ (place images listed in README)

## Features implemented (per assignment)
- Semantic HTML5 structure: header, nav, main, section, article, aside, footer.
- Responsive layout using Flexbox & CSS Grid.
- CSS variables, hover effects, transitions, and one keyframe animation.
- Forms with labels and accessible attributes.
- Images include alt attributes; skip link & aria-labels for accessibility.
- Breakpoints: desktop (≥1024px), tablet (~768px), mobile (≤480–768px).

## How to run
1. Clone or download the folder.
2. Open `index.html` in a browser (no server required).
